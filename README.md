# Desafio ETL - Santander Dev Week

## Descrição
Projeto ETL com dados de usuários obtidos via API mock, enriquecidos com mensagens geradas por IA e persistidos em arquivo JSON.

## Fluxo ETL
- Extract: Leitura de CSV e mock de API
- Transform: Enriquecimento dos dados com mensagens personalizadas
- Load: Escrita do resultado em JSON

## Como executar
```bash
pip install -r requirements.txt
python main.py
